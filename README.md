# Reality Check

> Reality Check tracks ambitious claims, examines competing explanations, evaluates available evidence, and records what remains unresolved.

Claims are easy. Verification is hard.

Reality Check is a public record of ambitious predictions, strategic assertions, market narratives, and technological promises. Each claim is tracked, pressured, and revisited as evidence emerges — not to prove people wrong, but to understand which ideas survive contact with reality.

---

## How It Works

Each audit follows the same structure:

1. **Claim** — the exact assertion, quoted from the source
2. **Alternative Explanations** — at least three competing interpretations
3. **Evidence** — what supports it, what cuts against it, and what's missing
4. **Status** — Supported / Unresolved / Currently Unresolvable
5. **Review Trigger** — the future event that would reopen the file
